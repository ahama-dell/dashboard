공유기 설정 페이지는 사내 보안을 위해 외부 인터넷에서는 접속이 불가능하며, 반드시 **회사 내부망(회사 와이파이나 사내 PC)**에 연결된 상태에서만 접속하실 수 있습니다.

사내 네트워크에 연결된 컴퓨터를 기준으로 접속하는 방법은 다음과 같습니다:

1단계. 내 공유기 접속 주소 알아내기 (Default Gateway)
컴퓨터마다 공유기의 주소가 다를 수 있으므로 명령어로 확인하는 것이 가장 정확합니다.

윈도우 검색창에 cmd (명령 프롬프트)를 입력해 실행합니다.
검은 창에 **ipconfig**를 입력하고 Enter를 누릅니다.
영어/한글 목록 중 기본 게이트웨이 (Default Gateway) 항목 옆에 있는 IP 주소를 찾습니다.
예: 192.168.0.1 또는 192.168.1.1 등
이 기본 게이트웨이 주소가 바로 공유기 설정 페이지의 주소입니다.
2단계. 브라우저로 접속 및 로그인
크롬 등 인터넷 브라우저 주소창에 방금 확인한 기본 게이트웨이 주소를 입력하고 접속합니다.
예시: 192.168.0.1 입력 후 엔터
공유기 로그인 화면이 나타나면 관리자 아이디와 비밀번호를 입력합니다.
iptime 초기 아이디/비밀번호 기본값: admin / admin
(만약 비밀번호를 모르는 경우, 회사의 전산 담당자에게 공유기 비밀번호를 확인하셔야 합니다.)
WARNING

※ 현재 외부망에 계시다면? 지금 외부 인터넷에 계시다면 공유기 페이지에 직접 들어갈 수 없습니다.

회사 내부에 있는 직원에게 위 작업을 대신 요청하시거나,
회사 내부 PC에 **원격 제어 프로그램(TeamViewer, Anydesk 등)**을 통해 원격으로 접속한 뒤 그 PC에서 공유기 페이지로 접속하여 설정해야 합니다.
작업 요약:

공유기 관리자 페이지는 사내망 내부에서만 접근 가능함을 고지하고, ipconfig 명령을 이용해 기본 게이트웨이(Default Gateway) IP 주소를 추출하는 법과 브라우저 접속 및 로그인 계정 분실 시 대처 방안(사내 PC 원격 제어 우회 접속)을 가이드했습니다.
